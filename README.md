# JOTI tracker web app


##### Dev guide for testing:


1. install npm packeges - `Npm i`
2. Start firebase emulators and vite - `npm run debug`
3. Wati intill you see a box givening you links to the emulator
4. open the auth link and add a test user (email, username, password)
5. Open the web app the link form vite
6. Log in with the acount you just made
7. Done!
